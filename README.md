# 🏆 Baddy Scoreboard

A simple and interactive scoreboard application for tracking scores in real-time. Ideal for sports events, games, and team competitions.

## 🚀 Features

- ✅ Change Player Names
- ✅ Increment/decrement scores
- ✅ Reset functionality
